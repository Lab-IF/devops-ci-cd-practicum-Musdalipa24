# Reflection - Praktikum DevOps Pertemuan 01

Melalui praktikum ini, saya mendapatkan pemahaman awal tentang konsep DevOps dan pentingnya integrasi antara proses development dan operations.

Saya berharap pada praktikum selanjutnya dapat:

1. Memahami alur kerja CI/CD secara lebih mendalam.
2. Mampu membuat dan mengelola Docker container secara mandiri.
3. Menggunakan Git secara lebih profesional dalam manajemen versi.
4. Mengintegrasikan aplikasi ke dalam pipeline otomatis.
5. Memahami praktik DevOps sesuai standar industri.

Praktikum ini menjadi langkah awal untuk membangun kesiapan dalam menghadapi kebutuhan industri teknologi modern.
